# TO RUN WEBSITE
## 1. YOU MUST RUN THE BACKEND

In root dir >>
- `export FLASK_APP=generate_bio.py`
- `python -m flask run`


## 2. YOU MUST RUN THE FRONTEND

In `frontend/`>>
- Must install npm.
- `npm install`
- `npm start`
