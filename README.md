# ArtificialBandBiography



# TO RUN WEBSITE
## 1. YOU MUST RUN THE BACKEND

In root dir >>
- You prob need to pip install stuff like flask and flask_cors. 
- `export FLASK_APP=generate_bio.py`
- `python -m flask run`


## 2. YOU MUST RUN THE FRONTEND

In `frontend/`>>
- You prob need npm to run this
- `npm install`
- `npm start`
